# Video Game Sales Prediction System

End-to-end machine learning project that predicts global video game sales and provides market strategy insights using historical sales data.

## Problem Statement
Predict the global sales of a video game based on platform, genre, publisher, region-wise sales, and release year.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn, XGBoost
- Matplotlib, Seaborn
- Streamlit

## ML Pipeline
- Data Cleaning & EDA
- Feature Engineering
- Model Training (Random Forest, XGBoost)
- Model Evaluation
- Web App Deployment

## Results
Achieved high prediction accuracy using ensemble models and hyperparameter tuning.
